
# 🌌 Shery.js Hero Animation Page

A stunning landing page built using **HTML**, **CSS**, **JavaScript**, and **Shery.js**. Includes background transitions, hero animations, and responsive design. This project demonstrates interactive visuals with layered effects and modern UI principles.

---

## 🚀 Features

- 🎥 Dynamic image background animations (using `Shery.imageEffect`)
- 🎯 Absolute-positioned hero section with sliding headings
- 🎨 Custom fonts and style transitions
- 🖼️ GSAP and Three.js-based image effects
- 📦 Modular and clean structure for easy extension

## 🔧 Installation & Run

1. Clone this repository:
   ```bash
   git clone https://github.com/Pritpatel11/shery-hero-animation.git
   ```

2. Open in VS Code or any editor.

3. **IMPORTANT**: Run with Live Server:
   > Do not open with `file://` protocol directly.

   Recommended:
   - Install **Live Server** extension in VS Code.
   - Right-click `index.html` → "Open with Live Server"

---

## ⚠️ Known Issues

- Image load errors may occur if file paths are incorrect or if images are missing.
- Use `window.onload` to ensure Shery effects load after all images.
